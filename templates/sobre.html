Sobre o EducAI Universitário

O EducAI Universitário é um assistente educacional desenvolvido em Python e Flask, integrado à API Google Gemini para oferecer respostas em linguagem natural sobre vestibulares, ENEM e processos seletivos. O projeto foi idealizado para:

  • Injetar contexto temporal e notícias atualizadas em tempo real;
  • Manter histórico de conversas e usar cache para otimizar desempenho;
  • Fornecer guia passo a passo para estudantes brasileiros;
  • Oferecer interface moderna, mobile-first e inspirada no TikTok.

Ferramentas e tecnologias utilizadas: Flask, Google Generative AI (Gemini), Tailwind CSS, python-dotenv, cachetools, feedparser.
